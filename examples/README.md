# examples/

Runnable examples — sample repositories, `charter.yaml` configs, and CI
snippets that show Charter in context.

**Empty today.** Worked examples currently live in the product docs:
[Quickstart](https://use-charter.dev/docs/quickstart), the
[how-to guides](https://use-charter.dev/docs), and the
[GitHub Action page](https://use-charter.dev/docs/ci/github-action). As
standalone example projects are added, each will live here in its own folder
with a README explaining what it demonstrates.
