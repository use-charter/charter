# Feature Playbook

## Entry
- Clear task or RFC
- Relevant context files loaded

## Freshness
- Inspect manifests and lockfiles
- Fetch latest official docs for touched tools or APIs

## Loop
1. Understand scope
2. Change the smallest viable slice
3. Verify with explicit commands
4. Update specs/docs/evals

## Stop
- unclear product semantics
- cross-cutting scope expansion
- docs or version mismatch
