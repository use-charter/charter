**Goal:** Implement the MCP Static Config Scanner (`AE-MCP-001`, `AE-MCP-002`, `AE-MCP-003`) over JSON MCP config files, so `charter doctor` flags unpinned MCP servers, untrusted remote origins, and remote servers with no auth declaration — completing milestone M1.3 (T1.3.1).

**Architecture:** One new self-contained rule package `internal/rules/mcp` discovers and parses MCP config files (`.mcp.json`, `.cursor/mcp.json`, `.vscode/mcp.json`, `mcp.json`) into a normalized `Server` model with `encoding/json` (stdlib), then runs three independent checks against that model. A new minimal `internal/config` package loads the trusted-remote allowlist from `charter.yaml`. All three rules emit `High` findings with `path:line` locations via the Slice 4 `Locations` contract. The scanner is deterministic, offline, and adds no findings when no MCP config exists (Charter's own repo keeps tracked MCP config absent, so dogfood stays at 100).

**Tech Stack:** Go 1.26.3, existing Cobra-based `charter doctor`, `findings`/`scoring`/`repository` seams, `encoding/json` (stdlib) for MCP files, `gopkg.in/yaml.v3` (new, latest — see Task 1) for `charter.yaml`, fixture repos under `testdata/repos/`, root verification via `moon run :check`.

**Grounding (verified 2026-05-31, live sources — do not trust from memory):**

- **MCP config shape:** root key `mcpServers` (object: name → config). Stdio servers: `command` (string), `args` (`string[]`), `env` (object). Remote servers: `type` (`"http"` | `"sse"`; SSE deprecated early 2026 in favor of HTTP), `url` (string), `headers` (object). Env expansion uses `${VAR}`, `${VAR:-default}`, `$VAR`. All major clients (Claude Code, Cursor, VS Code) share this structure; only the file path/root key varies (VS Code historically uses `servers` — parse it as a fallback alias).
- **OWASP MCP Top 10** (official, OWASP Foundation, v0.1 **beta** as of 2026): `MCP04` Software Supply Chain Attacks & Dependency Tampering (→ AE-MCP-001), `MCP09` Shadow MCP Servers (→ AE-MCP-002), `MCP07` Insufficient Authentication & Authorization (→ AE-MCP-003). The architecture doc's numbering is accurate. Cite as "OWASP MCP Top 10 (beta)".
- **MCP spec revision:** `2025-11-25` is the latest **stable** revision (the architecture doc reference is current). A major **release candidate `2026-07-28`** is locked (final ships 2026-07-28) and hardens OAuth (issuer mix-up, RFC 9207) and makes the core stateless — this affects the wire protocol, **not** static config files. AE-MCP-003 therefore checks for the *presence of an auth declaration* (revision-resilient), not specific OAuth field names. A follow-up to re-validate AE-MCP-003 against `2026-07-28` when it ratifies is tracked separately (not this slice).

## Go & architecture alignment (binding for implementation + review)

This plan encodes the project's idiomatic-Go and layering conventions (per the `golang-patterns` / `golang-testing` skills and the existing `internal/rules/*` code). Implementers and reviewers MUST hold to these:

- **Layering / separation of concerns:** pure parsing (`config.go`) → pure rule logic (`mcp001/002/003.go`, functions over `[]ConfigFile`) → filesystem orchestration (`mcp.go` `Run`) → pipeline wiring (`doctor/run.go`). The pure functions take data and return concrete structs, so they unit-test without touching disk or git.
- **Accept data, return concrete types; no speculative interfaces** (YAGNI). The rule checks return `[]findings.Finding`. Do not introduce interfaces until a second implementation needs them.
- **Errors are values — never swallow them.** `Run` returns `([]findings.Finding, error)` (mirroring `gosecrets.RunSecretRules`), and `run.go` propagates it. Wrap with context: `fmt.Errorf("parse %s: %w", path, err)`. **Fail fast** on unreadable/malformed MCP config or `charter.yaml` (consistent with the repo's "fail fast" constraint and the inventory/secrets error paths). No `_ =` discards.
- **No package-level mutable state.** Package-level `var`s are read-only (compiled regexes, runner/auth-name sets), matching `internal/rules/ci`.
- **Determinism:** every check sorts its findings (by evidence); `Run` discovers config paths in sorted order. Output is stable across runs (the `:check` gate and golden assertions depend on it).
- **Testing:** table-driven subtests with `t.Run`; `t.Helper()` in helpers; `t.TempDir()` for filesystem fixtures; `t.Parallel()` for independent unit tests; **add a fuzz test** on the JSON parser boundary (`FuzzParseConfigFile`) since it consumes untrusted input. Tests run under `-race` via `moon run :test`. Target ≥85% line coverage for `internal/rules/mcp` (the arch doc's M1.1 T1.1.6 bar).
- **Stdlib-first:** `encoding/json`, `net/url`, `strings` do the work; the only new dependency is `gopkg.in/yaml.v3` for `charter.yaml` (Task 1), added under the AGENTS.md "inspect manifest + verify latest" constraint.

```go
// internal/rules/mcp/config.go
package mcp

import (
	"encoding/json"
	"fmt"
	"sort"
	"strings"
)

// Server is a normalized MCP server entry from any supported client config.
type Server struct {
	Name    string
	Command string
	Args    []string
	Env     map[string]string
	Type    string // "http" | "sse" | "" (stdio)
	URL     string
	Headers map[string]string
	Line    int // 1-based best-effort line of the server key; 0 if unknown
}

// ConfigFile is one parsed MCP config file.
type ConfigFile struct {
	Path    string
	Servers []Server
}

// IsRemote reports whether the server is a remote (http/sse) endpoint.
func (s Server) IsRemote() bool {
	return s.URL != "" || s.Type == "http" || s.Type == "sse"
}

type rawServer struct {
	Command string            `json:"command"`
	Args    []string          `json:"args"`
	Env     map[string]string `json:"env"`
	Type    string            `json:"type"`
	URL     string            `json:"url"`
	Headers map[string]string `json:"headers"`
}

type rawConfig struct {
	MCPServers map[string]rawServer `json:"mcpServers"`
	Servers    map[string]rawServer `json:"servers"`
}

func parseConfigFile(path string, data []byte) (ConfigFile, error) {
	var rc rawConfig
	if err := json.Unmarshal(data, &rc); err != nil {
		return ConfigFile{}, fmt.Errorf("parse %s: %w", path, err)
	}

	entries := rc.MCPServers
	if len(entries) == 0 {
		entries = rc.Servers
	}

	names := make([]string, 0, len(entries))
	for name := range entries {
		names = append(names, name)
	}
	sort.Strings(names)

	cf := ConfigFile{Path: path}
	for _, name := range names {
		r := entries[name]
		cf.Servers = append(cf.Servers, Server{
			Name:    name,
			Command: r.Command,
			Args:    r.Args,
			Env:     r.Env,
			Type:    r.Type,
			URL:     r.URL,
			Headers: r.Headers,
			Line:    lineOfKey(string(data), name),
		})
	}
	return cf, nil
}

// lineOfKey returns the 1-based line of the first occurrence of "name" as a JSON
// key. Deterministic best-effort; returns 0 if not found.
func lineOfKey(text, name string) int {
	needle := `"` + name + `"`
	for i, line := range strings.Split(text, "\n") {
		if strings.Contains(line, needle) {
			return i + 1
		}
	}
	return 0
}
```

```go
// internal/rules/mcp/mcp001.go
package mcp

import (
	"regexp"
	"sort"
	"strings"

	"go.use-charter.dev/charter/internal/findings"
)

// exactVersion matches a pinned semver-ish version or digest.
var (
	exactVersionPattern = regexp.MustCompile(`^v?\d+\.\d+\.\d+(?:[-+][0-9A-Za-z.\-]+)?$`)
	digestPattern       = regexp.MustCompile(`^(?:sha256:[0-9a-f]{64}|[0-9a-f]{40})$`)
	packageRunners      = map[string]struct{}{"npx": {}, "bunx": {}, "uvx": {}, "pnpm": {}, "dlx": {}, "yarn": {}}
)

// classifyPackageSpec splits a package token into name + version and reports
// whether the version is exactly pinned. Unpinned: missing version, "latest"
// and dist-tags, semver ranges, and floating git refs.
func classifyPackageSpec(token string) (name, version string, pinned bool) {
	token = strings.TrimSpace(token)
	if token == "" {
		return "", "", false
	}
	if strings.HasPrefix(token, "github:") || strings.HasPrefix(token, "git+") || strings.Contains(token, "#") {
		return token, "", false // floating git ref
	}
	at := strings.LastIndex(token, "@")
	// at == 0 means the only '@' is the scope prefix -> no version present.
	if at <= 0 {
		return token, "", false
	}
	name = token[:at]
	version = token[at+1:]
	return name, version, isPinnedVersion(version)
}

func isPinnedVersion(v string) bool {
	v = strings.TrimSpace(v)
	if v == "" {
		return false
	}
	return exactVersionPattern.MatchString(v) || digestPattern.MatchString(v)
}

// packageTokenFromArgs returns the package spec token for a runner-based stdio
// server (npx/bunx/uvx/...). Returns ok=false when no package token applies.
func packageTokenFromArgs(command string, args []string) (string, bool) {
	base := command
	if i := strings.LastIndexAny(base, "/\\"); i >= 0 {
		base = base[i+1:]
	}
	if _, ok := packageRunners[base]; !ok {
		return "", false
	}
	for _, a := range args {
		if strings.HasPrefix(a, "-") { // skip flags like -y, --yes
			continue
		}
		if a == "dlx" || a == "exec" { // pnpm dlx / yarn dlx
			continue
		}
		return a, true // first non-flag arg is the package spec
	}
	return "", false
}

func checkPinning(files []ConfigFile) []findings.Finding {
	var result []findings.Finding
	for _, cf := range files {
		for _, s := range cf.Servers {
			token, ok := packageTokenFromArgs(s.Command, s.Args)
			if !ok {
				continue
			}
			if _, _, pinned := classifyPackageSpec(token); pinned {
				continue
			}
			result = append(result, findings.Finding{
				RuleID:      "AE-MCP-001",
				Severity:    findings.SeverityHigh,
				Category:    "MCP Safety",
				Summary:     "MCP server package is not pinned to an exact version (supply-chain risk, OWASP MCP04)",
				Remediation: "Pin the MCP server package to an exact version or digest instead of @latest, a semver range, or a floating git ref.",
				Evidence:    []string{cf.Path + ": server " + s.Name + " uses " + token},
				Locations:   []findings.Location{{Path: cf.Path, Line: s.Line}},
			})
		}
	}
	sort.Slice(result, func(i, j int) bool {
		return result[i].Evidence[0] < result[j].Evidence[0]
	})
	return result
}
```

```go
// internal/rules/mcp/mcp002.go
package mcp

import (
	"net/url"
	"sort"
	"strings"

	"go.use-charter.dev/charter/internal/findings"
)

func remoteHost(rawURL string) (string, bool) {
	u, err := url.Parse(strings.TrimSpace(rawURL))
	if err != nil || u.Host == "" {
		return "", false
	}
	return strings.ToLower(u.Hostname()), true
}

func isLocalHost(host string) bool {
	switch host {
	case "localhost", "127.0.0.1", "::1", "0.0.0.0":
		return true
	}
	return strings.HasSuffix(host, ".localhost")
}

func checkTrustedRemotes(files []ConfigFile, allow []string) []findings.Finding {
	allowed := map[string]struct{}{}
	for _, h := range allow {
		allowed[strings.ToLower(strings.TrimSpace(h))] = struct{}{}
	}

	var result []findings.Finding
	for _, cf := range files {
		for _, s := range cf.Servers {
			if !s.IsRemote() {
				continue
			}
			host, ok := remoteHost(s.URL)
			if !ok || isLocalHost(host) {
				continue
			}
			if _, trusted := allowed[host]; trusted {
				continue
			}
			summary := "Remote MCP server origin is not in the trusted-remote allowlist (OWASP MCP09 Shadow Servers)"
			if len(allow) == 0 {
				summary = "Remote MCP server origin cannot be verified — no charter.yaml mcp.trustedRemotes allowlist is configured (OWASP MCP09)"
			}
			result = append(result, findings.Finding{
				RuleID:      "AE-MCP-002",
				Severity:    findings.SeverityHigh,
				Category:    "MCP Safety",
				Summary:     summary,
				Remediation: "Add the host to charter.yaml mcp.trustedRemotes after review, or replace the server with a trusted origin.",
				Evidence:    []string{cf.Path + ": server " + s.Name + " -> " + host},
				Locations:   []findings.Location{{Path: cf.Path, Line: s.Line}},
			})
		}
	}
	sort.Slice(result, func(i, j int) bool { return result[i].Evidence[0] < result[j].Evidence[0] })
	return result
}
```

```go
package mcp

import (
	"sort"
	"strings"

	"go.use-charter.dev/charter/internal/findings"
)

// authHeaderNames are case-insensitive header keys that declare authentication.
var authHeaderNames = []string{"authorization", "x-api-key", "api-key", "x-auth-token"}

func hasAuthHeader(headers map[string]string) bool {
	for k, v := range headers {
		if strings.TrimSpace(v) == "" {
			continue
		}
		lk := strings.ToLower(strings.TrimSpace(k))
		for _, name := range authHeaderNames {
			if lk == name {
				return true
			}
		}
	}
	return false
}

func checkRemoteAuth(files []ConfigFile) []findings.Finding {
	var result []findings.Finding
	for _, cf := range files {
		for _, s := range cf.Servers {
			if !s.IsRemote() {
				continue
			}
			host, ok := remoteHost(s.URL)
			if !ok || isLocalHost(host) {
				continue
			}
			if hasAuthHeader(s.Headers) {
				continue
			}
			result = append(result, findings.Finding{
				RuleID:      "AE-MCP-003",
				Severity:    findings.SeverityHigh,
				Category:    "MCP Safety",
				Summary:     "Remote MCP server declares no authentication metadata (OWASP MCP07)",
				Remediation: "Declare an auth header (e.g. Authorization referencing an environment variable) for the remote MCP server, or switch to a local/trusted integration mode.",
				Evidence:    []string{cf.Path + ": server " + s.Name + " (" + host + ") has no auth header"},
				Locations:   []findings.Location{{Path: cf.Path, Line: s.Line}},
			})
		}
	}
	sort.Slice(result, func(i, j int) bool { return result[i].Evidence[0] < result[j].Evidence[0] })
	return result
}
```

```go
// internal/rules/mcp/mcp.go
package mcp

import (
	"fmt"
	"os"
	"path/filepath"
	"sort"

	"go.use-charter.dev/charter/internal/config"
	"go.use-charter.dev/charter/internal/findings"
	"go.use-charter.dev/charter/internal/repository"
)

// isMCPConfigPath reports whether p is an MCP config file Charter scans. Keep
// this list consistent with AE-SEC-002's MCP targets (future drift-guard candidate).
func isMCPConfigPath(p string) bool {
	switch p {
	case ".mcp.json", "mcp.json", ".cursor/mcp.json", ".vscode/mcp.json":
		return true
	}
	return false
}

// Run evaluates AE-MCP-001/002/003 across all MCP config files in the repo. It
// fails fast (returns a wrapped error) when a discovered MCP config or
// charter.yaml is unreadable or malformed, mirroring gosecrets.RunSecretRules.
func Run(root string, inv repository.Inventory) ([]findings.Finding, error) {
	var paths []string
	for _, p := range inv.Paths {
		if isMCPConfigPath(p) {
			paths = append(paths, p)
		}
	}
	if len(paths) == 0 {
		return nil, nil
	}
	sort.Strings(paths)

	files := make([]ConfigFile, 0, len(paths))
	for _, rel := range paths {
		// #nosec G304 -- rel is a fixed MCP config path from the tracked inventory.
		data, err := os.ReadFile(filepath.Join(root, filepath.FromSlash(rel)))
		if err != nil {
			return nil, fmt.Errorf("read %s: %w", rel, err)
		}
		cf, err := parseConfigFile(rel, data)
		if err != nil {
			return nil, err
		}
		files = append(files, cf)
	}

	allow, err := config.LoadTrustedRemotes(root, inv)
	if err != nil {
		return nil, err
	}

	var all []findings.Finding
	all = append(all, checkPinning(files)...)
	all = append(all, checkTrustedRemotes(files, allow)...)
	all = append(all, checkRemoteAuth(files)...)
	return all, nil
}
```
