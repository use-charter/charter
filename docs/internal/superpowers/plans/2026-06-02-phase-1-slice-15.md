**Goal:** ship the styled non-interactive `doctor` output + the opt-in `charter doctor -i` TUI + `charter explain <RULE>` + `charter doctor --rule` + `--color`/`--no-color`, via the `charm.land` v2 stack + `fang`. Offline/deterministic/no-LLM/no-network. Score formula (ADR-0008) unchanged.

Always run the FULL `mise exec -- moon run :lint`.

## Task ordering (build stays green)
deps (pin charm.land v2 + fang) → `internal/terminal` (capability + palette) → `internal/render/text` (styled + **containment contract test**) → `fang` wrap → `internal/explain` + `explain` command → `doctor --rule` + `--color`/`--no-color` → `internal/tui` (`doctor -i`) → docs (architecture §1.8 + command-surface, mirror regen) → final review + push.

## Task 1: dependencies
- [ ] `go get` and pin (verified 2026-06-02): `charm.land/bubbletea/v2@v2.0.7`, `charm.land/lipgloss/v2@v2.0.3`, `charm.land/bubbles/v2@v2.1.0`, `github.com/charmbracelet/fang@v1.0.0` (+ `charm.land/glamour@v1.0.0` only if MD is rendered in-TUI). `go mod tidy`; commit `go.mod`/`go.sum`. Confirm the build is green and binary size is noted.

## Task 2: internal/terminal (capability + palette)
- [ ] Capability detector: `isatty(stdout)`, `NO_COLOR`, `--no-color`/`--color` override, `TERM=dumb`, `COLORTERM`/`TERM` tier → {truecolor,256,16,mono}; OSC 8 support heuristic. Map the `DESIGN-TOKENS.md` semantic tokens → Lipgloss `AdaptiveColor` (light/dark) with per-tier degradation. Unit-test the **precedence matrix** (`--no-color` > `NO_COLOR` > `TERM=dumb` > `!isatty` > `COLORTERM`/`TERM`).

## Task 3: internal/render/text (styled non-interactive) + containment contract test
- [ ] Styled renderer for `doctor` matching the design refs (brand mark, `❯`, per-category rows, scan bars, severity pills, finding cards, score hero, key-hint footer) with **ASCII fallback** for mono/no-Unicode; OSC 8 links gated by color/TTY. Wire into `cmd/charter doctor` for the TTY path; plain text on non-TTY.
- [ ] **Containment contract test (load-bearing):** `--format {json,sarif,markdown}`, piped stdout, `--quiet`, and `NO_COLOR=1` emit **zero ANSI** and bytes identical to the pre-slice baseline (golden). Snapshot the styled path per color tier (forced-tier flag).

## Task 4: fang wrap
- [ ] Wrap the Cobra root in `fang` for styled help/usage/errors, `--version`, man pages — configured to honor `NO_COLOR`/TTY (no color on pipes). Verify `charter`/`charter --help`/error output; ensure no behavior change to `doctor`/`fix`/etc. exit codes.

## Task 5: charter explain + doctor --rule + color flags
- [ ] `internal/explain` over `internal/rules/catalog`; `charter explain <RULE>` (`--format text|json`); unknown rule → exit 2 with guidance. Unit-test every catalog rule resolves + JSON shape.
- [ ] `charter doctor --rule AE-XXX[,…]` scopes the scan to named rules. `--color auto|always|never` + `--no-color`. Tests.

## Task 6: internal/tui (charter doctor -i)
- [ ] Bubble Tea MVU master-detail browser over a single `doctor.Result`: header (brand + score + scorecard), findings list (`bubbles` list/table, group/sort by severity & category), detail viewport (id/severity/category/summary/evidence/remediation/path:line/auto-fix/OWASP/`explain` rationale), filter by severity/category, `/` search, vim+arrow nav, `tab` panes, `y` copy (OSC 52), `r` rescan, `?` help, `q` quit. Errors exit 2 if not a TTY.
- [ ] Model-level unit tests (`teatest`/message injection) for `Update` transitions (filter/search/select/rescan). No live-screen golden.

## Task 7: docs — architecture, command surface, mirror
- [ ] Architecture: amend §1.8 + command-surface text — add the interactive TUI surface; move `charter explain`, `--no-color`/`--color`, `doctor --rule` from the Phase 1.5 preview into the v1.0 surface. Regenerate the HTML mirror (`bun scripts/generate-doc-html.ts`; CF-2 `docs-html` gate). Update README current-state if needed.

## Task 8: final review + push
- [ ] `moon run :check` green; dogfood `charter doctor` (styled) = **100** and `charter doctor -i` renders; re-read the diff vs ADR-0024 (containment holds, non-TTY bytes unchanged, score formula untouched, no uninstalled-skill refs). Push the slice.

## Cross-task invariants
- Charm/ANSI **only** on the interactive/TTY path; non-TTY/`--format`/`NO_COLOR` byte-identical to baseline.
- Score formula (ADR-0008) unchanged; no new scan logic in renderer/TUI.
- No network, no LLM. Fonts are terminal-inherited (no typeface set). Use the installed design skills (`ui-ux-pro-max`, `design-taste-frontend`, `visual-design-foundations`, `interaction-design`).
