Scoring Reference

How the score is calculated

1
Start at 100, subtract for every failing rule
Blocker−20 pts each
High−10 pts each
Medium−4 pts each
Low−1 pt each
Score floors at 0 — it can never go negative.
↓ then apply hard ceilings
2
Apply the lowest applicable hard ceiling
Any Blocker present max score: 59
Raw secret (AE-SEC-001/002) max score: 49
Any rule unscanned / unknown max score: 79
If multiple ceilings apply, the lowest one wins.
⚠ AE-SUPPRESS-003 is informational — failing it does not deduct points or affect the score. All other rules follow the formula above.
