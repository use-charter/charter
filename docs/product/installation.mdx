---
title: "Installation"
description: "Install Charter on macOS, Linux, or Windows."
---

Charter ships as a single static binary with no runtime dependencies. Pick the install path that fits your workflow.

## Homebrew (macOS and Linux)

The fastest path on macOS and most Linux distributions:

```bash
brew install use-charter/tap/charter
```

Homebrew handles version management and upgrades (`brew upgrade charter`).

## Direct binary download

Pre-built binaries are published to GitHub Releases for the following platforms:

| OS | Architecture | Asset pattern |
|---|---|---|
| macOS | x86_64 | `charter_darwin_amd64.tar.gz` |
| macOS | Apple Silicon | `charter_darwin_arm64.tar.gz` |
| Linux | x86_64 | `charter_linux_amd64.tar.gz` |
| Linux | ARM64 | `charter_linux_arm64.tar.gz` |
| Windows | x86_64 | `charter_windows_amd64.zip` |
| Windows | ARM64 | `charter_windows_arm64.zip` |

Download from:

```
https://github.com/use-charter/charter/releases/latest
```

**Steps (macOS / Linux):**

```bash
# Download and extract (replace <version> and <platform> as appropriate)
curl -L https://github.com/use-charter/charter/releases/download/v<version>/charter_<platform>.tar.gz | tar xz

# Move the binary to a directory on your PATH
sudo mv charter /usr/local/bin/charter
```

**Steps (Windows):**

Download the `.zip` from the releases page, extract `charter.exe`, and move it to a directory that is on your `PATH`.

Every release asset is signed with cosign keyless signing and accompanied by SLSA L3 provenance. Verify a release:

```bash
cosign verify-blob \
  --certificate charter_linux_amd64.tar.gz.cert \
  --signature charter_linux_amd64.tar.gz.sig \
  charter_linux_amd64.tar.gz
```

## go install

If you have a Go toolchain available (Go 1.21 or later required):

```bash
go install go.use-charter.dev/charter/cmd/charter@latest
```

The binary is placed in `$(go env GOPATH)/bin`. Make sure that directory is on your `PATH`.

## Build from source

```bash
git clone https://github.com/use-charter/charter
cd charter
go build -o charter ./cmd/charter
```

The resulting `charter` binary can be placed anywhere on your `PATH`. Building from source requires Go 1.21 or later.

---

## Verify the install

After any install path, confirm the binary is working:

```bash
charter version
```

Expected output includes the version number, the commit hash it was built from, and the platform:

```
charter v1.0.0 (abc1234) darwin/arm64
```

If the command is not found, ensure the directory containing the binary is on your `PATH` and open a new shell session to pick up the change.
