# ADR-0010 Trademark Clearance

- Status: Pending external clearance
- Context: Charter architecture makes trademark clearance and domain acquisition a hard gate before public launch, public branding lock-in, and external promotion.
- Decision: Keep local repo and code references on `Charter` during foundation work, but treat legal name clearance and domain acquisition as an external owner decision that must resolve to `CLEARED` or `RENAMED` before launch.
- Consequences: Repo-local Phase 0 foundation work may proceed. Public launch, public repo promotion, and permanent brand/domain commitments must stay blocked until this ADR is updated with the external outcome.
- Exit criteria: update this ADR to `CLEARED` with approved domain evidence, or `RENAMED` with the replacement product name and follow-up rename plan.
