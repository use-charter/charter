# Embedded report fonts — licenses & attribution

The Charter HTML report (`charter report --format html`, ADR-0025) embeds the brand
typefaces directly in its CSS as base64 `woff2` `data:` URIs so the report stays
**100% self-contained and offline** — no CDN, no remote fetch at render or view time.

The shipped artifact is `../fonts.css` (generated by
`scripts/generate-report-fonts.ts`, which fetches the Latin-subset `woff2` once from
the Google Fonts CSS2 API at author time). The `woff2` payloads are not stored as
separate files in the repo — they live inline in `fonts.css`.

All four families are licensed under the **SIL Open Font License, Version 1.1**
(embeddable). The full, verbatim license text for each family is vendored alongside
this file:

|---|---|---|---|---|
| Ruda | 400, 500, 700, 800 | `--font-site` / `--font-audit` | Copyright 2019 The Ruda Project Authors (Omnibus-Type) | [`Ruda-OFL.txt`](./Ruda-OFL.txt) |
| Atkinson Hyperlegible Mono | 400–700 (variable) | `--font-code` | Copyright 2020–2024 The Atkinson Hyperlegible Mono Project Authors (Braille Institute of America) | [`AtkinsonHyperlegibleMono-OFL.txt`](./AtkinsonHyperlegibleMono-OFL.txt) |

| Share Tech | 400 | `--font-accent` | Copyright 2012 The Share Tech Project Authors (Carrois Apostrophe) — Reserved Font Name "Share" | [`ShareTech-OFL.txt`](./ShareTech-OFL.txt) |

Only the `latin` subset of each face is embedded; every CSS font stack still ends in
a system fallback, so glyphs outside the subset and any font-load failure degrade
gracefully to the OS font (DESIGN-TOKENS.md, ADR-0001 offline-first).

To refresh the embedded fonts (requires network):

```sh
bun scripts/generate-report-fonts.ts
go test ./internal/render/html/... -run TestGolden -update   # regenerate the golden fixture
```
