# Code of Conduct

## Our standard

Charter adopts the **Contributor Covenant, version 2.1** as its code of conduct.
Everyone participating in the project — issues, pull requests, discussions, and
any other project space — is expected to follow it. In short: be respectful and
constructive, assume good faith, and help keep this a welcoming community for
people of all backgrounds and experience levels.

The full text is canonical and available here:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

A copy can also be generated with `contributor-covenant` tooling or read on the
Contributor Covenant site above.

## Scope

This Code of Conduct applies within all project spaces and also applies when an
individual is officially representing the project in public spaces.

## Reporting

Report unacceptable behavior privately to the maintainers at
**conduct@use-charter.dev**. Reports are handled confidentially. For security
vulnerabilities, use a private advisory instead (see [SECURITY.md](SECURITY.md)).

## Enforcement

Maintainers are responsible for clarifying and enforcing this standard and may
take any action they deem appropriate, up to and including a temporary or
permanent ban, following the Contributor Covenant's enforcement guidelines.
